MiniML – Application de prédiction nutritionnelle

MiniML est une application web de démonstration permettant d'exploiter un modèle de Machine Learning pour estimer des indicateurs nutritionnels à partir de caractéristiques physico-chimiques de matières premières agricoles.

Fonctionnalités principales

  • Modèle XGBoost multi-sorties pour prédictions nutritionnelles simultanées
  • Encodage automatique des variables catégorielles (OneHotEncoder sécurisé)
  • Imputation automatique des valeurs manquantes à partir des moyennes Feedtables
  • Sélection dynamique des intervalles de confiance selon la complétude des données
  • Historique intégré des entrées et sorties de prédiction
  • Export CSV complet des prédictions (inputs + outputs + indicateur d'imputation)
  • Visualisation des performances du modèle (MAE, RMSE, R²)

Navigation

Utilisez le menu à gauche pour :

  • Prédiction : saisir des données, générer une estimation et consulter l’historique intégré
  • Model Info : consulter les performances globales et la structure du modèle
  • Diagnostics : analyser les résidus et la qualité prédictive
Cette application illustre l’intégration d’un pipeline Data Science complet (prétraitement, modélisation, interprétation et export des résultats) dans une interface web interactive.
